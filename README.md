# PAH-Pressure-Sensor
Contains the code used to read the pressure inside of a bioengineered construct used to simulate the pulmonary artery of patients with pulmonary arterial hypertension. This code was written using the Arduino IDE 2.0 and is expected to be used with an Arduino or a board
that natively supports C/C++. The sensor used in this project is the Honywell HSCMANV160MG2A3.
